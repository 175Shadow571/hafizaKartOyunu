# Temel bir hafıza kart oyunu
Oyun basit oyun toplam 7 bölümden oluşmaktadır her bölümde 10 karakter vardır ve her birini tek kez tıklamayla bulmanız lazım bir karaktere 2 kez tıklarsanız oyun başa döner

## Çalıştırma için
backend node proxy.server.js
frontend npm run dev

resimlerin gele bilmesi için superheroapi.com den api key gereklidir aksi taktirde isimler ile oynanır
